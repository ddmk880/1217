    关于地图的制作我没有太仔细地进行研究，这里只是简单的说明。运行游戏目录下的“地图编辑器.bat”文件以激活游戏内置的地图编辑器，首先你点击地图编辑器菜单“地图”-->“设置”，必须设置一个背景地图，然后就可以调用“舰船调用窗口”里的东西了，用鼠标把你需要的粘帖在空白地图上就可以了，右键可以编辑/制造。建议在设置完背景地图后测试地图（当然这时候运行的游戏中只有背景地图），把“调试窗口”的“无敌”勾选，这样可以避免因为你长时间未进行地图测试，游戏自动运行DEMO，而可能发生地图背景更换的情况，这可是我的经验之谈喔^△^。
    另外要说明一点的是，经过编辑/制造的舰船只在测试地图时有效。如果你删除了部分舰船，想要恢复怎么办？只需删除游戏目录下的“SHIPDEFS.GLB”这个文件就要以了，此文件在下次激活地图编辑器时会自动生成，所以不用担心啦！“调试窗口”可以适时地对游戏进行控制，相当于一个游戏的秘技工具。你可以运行游戏目录下的“调试.bat”这个文件以作为秘技工具使用，这下你可以无敌了，还怕不能过关吗？
    制作好的 MAP 格式的地图文件该怎么使用呢？很简单，只要在游戏的“文件”菜单里选择“加载地图”即可。

    ※      关于游戏连网　      ※
    ※                          ※
    ※※※※※※※※※※※※※※※
    
    主机点击游戏菜单“连网”－“创建”，输入要创建的游戏名称和你在游戏中的名字，连接类型按默认，然后点击“确定”，等待另一个人加入后，按“F2”键就可以开始连网玩了。
　　副机点击游戏菜单“连网”－“加入”，输入你在游戏中的名字，连接类型默认即可，点击确定后输入对方主机的 IP 地址或不输入让电脑自动查找。搜索到主机后会在“加入游戏对话框”里出现主机名称，选择后点击“确定”按钮，等待主机开始游戏(必须由主机开始)后按“F2”开始游戏。

    如何获得本机 IP 地址？点击游戏对话框中的“获取 IP 地址”所得到的通常都不对(原版如此，非汉化之故)。
    正确的方法有很多，用百度等搜索引擎在地址栏里输入关键字“IP查询”，你可以查到很之方法，现列举一些方法如下：
    1、在线获取。在浏览器地址栏里输入网址 http://www.oicq88.com/ip/ 在页面上就会显示你的 IP地址了
    2、或者使用一些可查询 IP 地址的工具软件，追捕、无处藏身、带显 IP 功能的QQ等等。
    3、使用系统自带的 IP 查询工具。
       A：win98和win2000系统在“运行”栏里输入：winipcfg
       B：winxp系统在“运行”栏里输入：cmd，之后再输入 ipconfig，显示在“IP ADDress”后面的就是你的 IP 地址了。
       C：使用 NetMeeting 查询。查看 NetMeeting 帮助菜单里的关于会看到 IP 地址。
    4、使用傲游浏览器Maxthon（旧版称MyIE2）,将“选项-->状态栏-->显示本机IP地址”打勾，即可在状态栏里看到本机IP。
